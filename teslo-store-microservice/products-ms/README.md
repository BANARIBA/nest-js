# Steps to run this microservice project
1. Clone this repository
2. Copy the __.env.template__ and paste renaming the name to __.env__
3. Set the environment vars in: __.env__
4. Run docker compose using: ```docker compose up -d```
5. Install microservice dependencies using: ```npm i```
6. Run this project: ```npm run start:dev```
