# Teslo shop app

## Dependencies
1. [Nest Config](https://docs.nestjs.com/techniques/configuration)
2. [Validator & Transformer](https://docs.nestjs.com/techniques/validation)
3. [TypeOrm](https://docs.nestjs.com/techniques/database)

## Steps to run this project
1. Clone this repo
2. Install dependencies ```npm install```
3. Copy __.env.template__ file and paste and rename with __.env__, then set the environment vars
4. Run project  ```npm run start:dev```
